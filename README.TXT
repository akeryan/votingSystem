1) git clone the repository
2) run 'npm install'
3) run 'npx hardhat compile'
4) run 'npx hardhat run scripts/deploy.js --network rinkeby'

Testing:
1) run 'npx hardhat test'

Check coverage:
1) run 'npx hardhat coverage'

Tasks:
1) run 'npx hardhat' to check all the available Tasks
2) run 'npx hardhat <task> <options> --network rinkeby
